Bank Marketing KPI Analysis (Excel Project)

Project Overview

This project focuses on analyzing a bank marketing campaign dataset using Microsoft Excel to measure campaign performance through Key Performance Indicators (KPIs).

The goal is to identify customer segments, campaign effectiveness, and factors influencing successful product subscriptions.

The analysis is entirely performed in Excel using data cleaning, formulas, pivot tables, and dashboard visualizations.

Objective:


Measure campaign success using KPI metrics

Analyze customer behavior

Identify high-conversion segments

Support data-driven marketing decisions

Dataset Description

Records: 45,112 customers

Attributes: 28 columns

Target Variable: response (yes / no)

Key Columns:


Customer Info: age, marital, job, education

Financial Info: salary, balance, loan, housing, default

Campaign Info: contact, month, duration, campaign

Output: response

Data Cleaning & Preparation (Excel)

Handled missing values in age using average

Converted call duration from text (261 sec) to numeric

Split combined columns (jobedu, month)

Created helper columns:


age_group

balance_type

response_num

Key Performance Indicators (KPIs)

KPI	Description:


Total Customers	Total customers contacted

Total Subscriptions	Customers who said YES

Conversion Rate (%)	YES / Total Customers

Avg Call Duration (YES)	Successful calls

Avg Call Duration (NO)	Unsuccessful calls

Best Performing Month	Highest subscriptions

Best Contact Method	Highest conversion

Age Group Conversion	Segment-wise success

Loan Impact	Loan vs subscription

Key Insights:


Middle-aged customers showed higher conversion rates

Longer call durations significantly increased success

Customers without personal loans subscribed more

Certain months performed better for campaigns

Direct contact methods were more effective
